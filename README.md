# ASCII Art Web

## Description
ASCII Art Web is a Go-powered web application that allows users to convert text into ASCII art using multiple banner styles: **standard**, **shadow**, and **thinkertoy**. Users can enter text, choose a banner style, and view the resulting ASCII art directly in their browser. The project demonstrates the integration of Go backend, HTTP server, and dynamic HTML templates.
